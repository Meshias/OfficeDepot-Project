2.- Carpeta Libros - Angular
	2.1.- Ejecutar el servidor de Angular desde el Simbolo del Sistema o Editor con Terminal
		-Simbolo del Sistema
			-Abrir el Simbolo del Sistema como ADMINISTRADOR
			-Ir a la direccion del Proyecto de Angular (cd Libros si esta en el escrito de preferencia)
			-Ejecutar el Comando ng serve para Iniciar el Proyecto.

		-Editor de Codigo (Visual Studio Code)
			-Descargado el Proyecto, abrir el editor y abrir la carpeta del Proyecto desde el Editor
			-Ir a la Opcion Terminal en la parte superior, seleccionar "Nueva Terminar"
			-En la parte Inferior derecha, cambiar la opcion de PowerShell a Command Prompt
			-Ejecutar el Comando ng serve para Iniciar el Proyecto.

	2.2.- Entrar a la Direccion Local de Angular: localhost:4200 para visualizar el proyecto
